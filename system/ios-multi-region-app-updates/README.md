# iOS: 减少不同地区账号 App 更新时的密码提示

## 背景
如果 iPhone 上安装过来自不同地区 Apple 账号的 App，更新时经常会反复弹出对应账号的密码输入提示，流程比较打断。

一个实用做法是把这些地区账号都添加到系统的邮件账户里，后续更新时通常可以减少频繁输入密码。

## 使用前提
- 你在 iPhone 上使用过多个地区的 Apple 账号下载 App
- 你能正常登录这些账号
- 系统自带“邮件”App可用

## 如何使用
1. 打开 iPhone：`设置 -> 邮件 -> 账户 -> 添加账户`
2. 选择 `iCloud`，分别登录你需要的不同地区账号
3. 完成后回到 App Store 更新页面，再尝试更新应用

补充：
- 这是“减少频繁输入密码”的方式，不是完全消除所有验证
- 如果仍有个别 App 提示登录，按提示完成一次验证后一般会好很多
