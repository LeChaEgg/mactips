# macOS: 一行命令调整菜单栏图标间距

## 背景
当菜单栏图标过多时，默认间距会占用不少空间。可以通过 `defaults` 一行命令调整图标之间的间距，让菜单栏更紧凑。

## 使用前提
- 使用 macOS
- 有终端访问权限
- 调整后需要退出登录再重新登录才能生效

## 如何使用
1. 打开终端
2. 粘贴并执行以下命令：

```bash
defaults -currentHost write -globalDomain NSStatusItemSpacing -int x
```

3. 把命令最后的 `x` 替换为你想要的间距值
- 最小值是 `0`
- 建议从 `5` 开始尝试
4. 退出当前登录会话并重新登录系统

重新登录后即可看到菜单栏图标间距变化。
